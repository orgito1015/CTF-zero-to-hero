Tools & Setup

You don't need every tool. Start with the essentials for your category of interest, then expand as you tackle more challenges. All of these are free and open source.

Essential Tools

Tool Category Description Install Command
CyberChef Crypto / Encoding Swiss-army knife for encoding, decoding, and data transformation. Works entirely in the browser. Browser — no install needed
Burp Suite Community Web Intercept, inspect, and modify HTTP/HTTPS requests. Industry-standard web testing proxy. apt install burpsuite or portswigger.net
Ghidra Reverse Engineering NSA's free, open-source reverse engineering framework. Decompiles x86, ARM, and many other architectures. ghidra-sre.org (manual download)
Wireshark Forensics Graphical network packet analyzer. Capture and inspect network traffic in real time. apt install wireshark
pwntools Pwn Python library for exploit development. Simplifies writing shellcode, ROP chains, and remote connections. pip3 install pwntools
binwalk Forensics Firmware and file analysis tool. Extracts embedded files and detects hidden data in binary files. apt install binwalk
hashcat Crypto GPU-accelerated password and hash cracking tool. Supports MD5, SHA1, SHA256, bcrypt, and hundreds more. apt install hashcat
GDB + pwndbg Pwn GNU Debugger with the pwndbg plugin — adds CTF-friendly views for stack, registers, and memory. See setup instructions below
SQLMap Web Automated SQL injection detection and exploitation. Supports all major database backends. pip3 install sqlmap
exiftool Forensics Read and write metadata from virtually any file format. Flags often hide in EXIF comments. apt install exiftool

One-Click Setup

Install everything at once with the setup script included in this repository:

bash 
git clone https://github.com/orgito1015/CTF-zero-to-hero
cd CTF-zero-to-hero
bash scripts/setup.sh

The setup script installs common tools available via apt and pip3. Run on Kali Linux or Ubuntu for best results.

GDB + pwndbg Setup

bash 
# Install GDB
apt install gdb

# Install pwndbg plugin
git clone https://github.com/pwndbg/pwndbg /opt/pwndbg
cd /opt/pwndbg && ./setup.sh

# Launch GDB with pwndbg
gdb ./vuln

Recommended Environment

🐉 Kali Linux (Recommended)

The gold standard for CTF and security work. Comes pre-loaded with hundreds of tools.

  • Pre-installed: Burp Suite, Wireshark, binwalk, exiftool, hashcat, GDB, nmap, and many more
  • Actively maintained rolling-release distribution
  • Available as a VM image, ISO, WSL package, or Docker container
  • Large community with extensive documentation
  • Download: kali.org/get-kali

🐧 Any Linux / WSL2

Ubuntu, Debian, or Arch Linux work great too. WSL2 on Windows is a solid option.

  • Ubuntu 22.04 LTS is a reliable, well-supported choice
  • WSL2 (Windows Subsystem for Linux) is surprisingly capable for CTF work
  • Use Docker for isolated challenge environments
  • Virtual machines (VirtualBox, VMware) keep your host OS safe
  • macOS with Homebrew works for most tools (except some Linux-specific ones)
