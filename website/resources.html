Learning Resources

🌐 Web Exploitation

  • PortSwigger Web Academy — Free, hands-on labs covering every web vulnerability. The best web security learning resource available.
  • OWASP Top 10 — The definitive list of the ten most critical web application security risks. Essential reading.

🔐 Cryptography

  • CryptoHack — Interactive crypto challenges with a CTF feel. Covers modern cryptography from RSA to elliptic curves.
  • Cryptopals — Classic crypto challenges that teach you to break real cryptographic systems through hands-on coding.

💥 Pwn / Binary Exploitation

  • pwn.college — ASU's free binary exploitation course. Structured curriculum from fundamentals to advanced techniques.
  • ROP Emporium — Purpose-built challenges to teach Return-Oriented Programming. Best resource for learning ROP chains.

🔍 Reverse Engineering

  • reversing.kr — Classic RE challenges at multiple difficulty levels. Good for practicing static and dynamic analysis.
  • crackmes.one — Community-submitted crackme binaries with difficulty ratings. Thousands of targets across multiple platforms.

🕵️ Forensics

  • 13Cubed YouTube — Excellent free video tutorials on digital forensics, memory analysis, and CTF forensics techniques.
  • AboutDFIR — Curated resource hub for digital forensics and incident response. Tools, training, and write-ups.

Cheatsheets

bash — First Commands 
# Identify file type
file challenge.bin

# Search for readable strings
strings challenge.bin | grep -i flag

# Extract embedded files
binwalk -e challenge.bin

# Check metadata
exiftool challenge.png

# Hex dump first 32 bytes
xxd challenge.bin | head -4

# Find flag pattern in directory
grep -r "CTF{" . 2>/dev/null
bash — Web Quick Hits 
# Check HTTP headers
curl -I http://target.ctf/

# Grab robots.txt
curl http://target.ctf/robots.txt

# SQLi quick test
curl "http://target.ctf/login?id=1'"

# Directory brute-force
gobuster dir -u http://target.ctf/ \
  -w /usr/share/wordlists/dirb/common.txt

# SQLMap one-liner
sqlmap -u "http://target.ctf/?id=1" \
  --batch --dump
bash / python — Crypto Quick Hits 
# Base64 decode
echo "Q1RGe...}" | base64 -d

# Hex decode
echo "4354467b..." | xxd -r -p

# ROT13
echo "text" | tr 'A-Za-z' 'N-ZA-Mn-za-m'

# Python: URL decode
python3 -c "import urllib.parse; \
  print(urllib.parse.unquote('%43%54%46'))"

# Python: bytes to int
python3 -c "print(int.from_bytes(b'CTF', 'big'))"

# Crack hash
hashcat -m 0 hash.txt rockyou.txt
